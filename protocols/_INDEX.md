# AI PROTOCOL INDEX

## Core Principles

1. **Integrity First** : Every change is documented and audited.
2. **Atomic History** : Single-purpose commits only.
3. **Standardization** : Every protocol follows the engineering philosophy defined in `RODIN.md`.
